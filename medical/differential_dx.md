# 鑑別診断フレームワーク

## 思考順序
1. **Killer Disease**（見逃すと致命的）を最初に除外
2. **Common Disease**（頻度の高い状態）を次に検討
3. **Rare but Must Not Miss**（稀だが重要）を最後に確認

## 記述形式（必須）
- 「〇〇が当てはまるなら → 状態A（確認方法：△△）」
- 「〇〇がないなら → 状態Bを低確率と考えられる」
- 「〇〇＋〇〇が揃うなら → 状態Cの可能性が高い」

## VINDICATE フレームワーク（系統的鑑別）
| 頭文字 | カテゴリ | 例 |
|-------|---------|---|
| V | Vascular（血管） | 脳梗塞・心筋梗塞・肺塞栓 |
| I | Infectious（感染） | 肺炎・髄膜炎・敗血症 |
| N | Neoplastic（腫瘍） | 悪性腫瘍・リンパ腫 |
| D | Degenerative（変性） | 変形性関節症・椎間板ヘルニア |
| I | Idiopathic（特発性） | IBS・線維筋痛症 |
| C | Congenital（先天性） | 先天性心疾患・代謝異常 |
| A | Autoimmune（自己免疫） | SLE・関節リウマチ・IBD |
| T | Traumatic（外傷） | 骨折・打撲・裂傷 |
| E | Endocrine/Metabolic（内分泌・代謝） | 糖尿病・甲状腺疾患・電解質異常 |

## 患者属性別の注意点
- **小児（15歳未満）**: アスピリン禁忌（Reye症候群）、体重ベース用量、熱性けいれん・川崎病を念頭に
- **高齢者（75歳以上）**: 非定型症状（無熱性感染・無痛性心筋梗塞）、転倒リスク、多剤併用（ポリファーマシー）
- **妊婦**: 安全性未確立薬の明示、子宮外妊娠・子癇・HELLP症候群を除外
- **免疫抑制者**: 日和見感染・非定型症状を考慮
