# EBM評価基準（GRADEシステム準拠）

## エビデンスグレード
| グレード | 根拠 | 推奨の確信度 |
|---------|------|------------|
| A | ランダム化比較試験（RCT）のメタ解析・複数のRCT | 強い推奨 |
| B | 単一のRCT・コホート研究・観察研究 | 条件付き推奨 |
| C | 症例対照研究・症例報告 | 弱い根拠 |
| D | 専門家意見・経験・病態生理からの推論 | 非常に弱い根拠 |

## 引用源の優先順位（日本語ユーザ向け）
1. 日本の学会ガイドライン（日本内科学会・各専門学会）
2. Cochrane Review
3. 国際ガイドライン（WHO・AHA・ESC等）
4. UpToDate / BMJ Best Practice
5. 教科書・専門家意見

日本ガイドラインと国際ガイドラインで推奨が異なる場合は両者を明示し差異を注記する。

## 引用なし主張の処理
引用源が明示できない推奨は integrator エージェントが削除する。
「一般的に言われている」等の曖昧な表現は使用しない。

## 不確実性の明示
- 各鑑別候補に「確信度（高/中/低）」を付与
- 追加情報があれば確信度が変わる点を明記
- 「現時点の情報のみに基づく暫定的な考え」であることを注記

## QAルーブリック（全項目 pass 必須）
| 項目 | Pass 条件 |
|-----|----------|
| red_flag_checked | 00-safety_net が "clear" を確認済み |
| differential_count | 鑑別候補が3つ以上 |
| evidence_grade_ab | グレードA/B推奨が1つ以上 |
| patient_factors | 年齢・性別・妊娠可能性への言及あり |
| home_care_present | 家庭対処法が具体的に記載済み |
| disclaimer_ready | 免責事項フラグがon |

全項目 pass → qa_status: "pass"
1項目でも fail → qa_status: "retry"（max 2回）
2回 retry 後も fail → qa_status: "needs_review"
