# トリアージロジック

## 緊急度スコア定義
| スコア | 区分 | 対応目安 |
|-------|------|---------|
| 5 | 即時受診（生命危機） | 今すぐ119番・ER |
| 4 | 当日受診必須 | 救急外来・本日中 |
| 3 | 数日以内受診 | かかりつけ医・3日以内 |
| 2 | 様子見可・悪化で受診 | 自宅療養・1週間目安 |
| 1 | 経過観察 | ホームケアで対応可 |

## 必ず構造化して収集する情報
| 項目 | 内容 |
|-----|------|
| 誰が | 年齢・性別・妊娠可能性 |
| 主訴 | 症状の種類・部位・性状 |
| 発症 | いつ・どのように始まったか |
| 経過 | 増悪・軽快・変動するか |
| 随伴症状 | 発熱・嘔吐・下痢・出血等 |
| 既往歴 | 慢性疾患・手術歴 |
| 内服薬 | 現在の服薬（サプリ含む） |
| アレルギー | 薬・食物 |
| 職業・渡航歴 | 感染症・職業病の除外に使用 |

## 追加質問が必要な場合
以下のいずれかが不明な場合は status: "need_more_info" を返す：
- 年齢・性別が不明
- 発症時期が全く不明（「ずっと前から」等）
- 主訴が1語のみで詳細不明（「痛い」「体調不良」等）

## Working Hypothesis の作り方
1. Killer Disease（見逃すと致命的・頻度が低くても必ず考える）
2. Common Disease（頻度が高い・確率的に最もあり得る）
3. Rare but Must Not Miss（稀だが見逃すと重篤な転帰）
各候補に根拠（Supporting features）と除外根拠（Against features）を付記。
